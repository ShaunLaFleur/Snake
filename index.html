Snake

A HTML, CSS & javascript project

Difficulty: Medium

Rules

  • Press any button to start the game.
  • Control the snake with your arrow keys.
  • Running into walls simply spits you out the other side!
  • Be careful not to take a bite out of yourself, if you do, you lose!
  • Eat the apple(green square) to add one segment to your snake. See how long you can become!.
  • You gain 2 points per snake length when eating an apple.
  • Web application is currently under development and is thus incomplete. Game does function, however, there is no winning condition, point counter or timers yet.
